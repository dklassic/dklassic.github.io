# 所以《終噬虛空》是款怎樣的作品？

《終噬虛空》是一款短篇的實驗性增量遊戲，以使用遊戲機制講述故事為主設計而成。遊戲機制是以《踩地雷》為靈感的體力管理遊戲循環。

# 哪裡可以買到《終噬虛空》？

《終噬虛空》即將推出在：

- [Steam](https://store.steampowered.com/app/2589050/)

# 最低系統配備是什麼？

說真的只要你的電腦比 Steam Deck 還強的話就保證跑得順。至於 Steam Deck 上這款遊戲的目標是 800p/90FPS。

# 《終噬虛空》對手把有支援嗎？

用手把玩，只用滑鼠玩，都可以！不過沒辦法只用鍵盤玩就是。

# 《終噬虛空》支援雲端存檔嗎？

有的！需要的話也可以在 `文件/Saved Games/UltraVoid` 找到本機的存檔。
